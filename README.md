#### In order to run
```sh
> git clone "https://github.com/Mahesh-Ranaweera/TODO"
> cd TODO
> npm electron-prebuilt --save-dev
> npm start
```
#### to Build
```sh

--- Windows build
> npm run build-win

---Linux build
> npm run build-linux

```
